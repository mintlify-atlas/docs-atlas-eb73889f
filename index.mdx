<div className="mt-16 mb-16 max-w-5xl mx-auto px-6">
  <h2 className="text-3xl font-bold text-gray-900 dark:text-white mb-4">Quick Start</h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">Get your EV charging infrastructure up and running in minutes</p>
  
  <Steps>
    <Step title="Install Prerequisites">
      Install required dependencies for building EVerest Core on your system.
      
      ```bash
      # Ubuntu/Debian
      sudo apt-get install -y cmake build-essential git python3 libboost-all-dev
      ```
      
      <Note>
        EVerest requires CMake 3.16+, a C++17 compiler, Python 3, and Boost libraries.
      </Note>
    </Step>
    
    <Step title="Clone and Build">
      Clone the repository and build the framework with CMake.
      
      ```bash
      git clone https://github.com/EVerest/everest-core
      cd everest-core
      cmake -B build -DCMAKE_INSTALL_PREFIX=./dist
      cmake --build build --target install
      ```
      
      The build process compiles all modules and creates run scripts for various configurations.
    </Step>
    
    <Step title="Run Your First Configuration">
      Start a simulated charging station using one of the provided configurations.
      
      ```bash
      # Run software-in-the-loop AC charging simulation
      ./build/run-scripts/run-sil.sh
      ```
      
      <Accordion title="Expected output">
        You'll see modules starting up and connecting via MQTT. The simulation includes a complete charging workflow with authentication, energy management, and OCPP communication.
      </Accordion>
    </Step>
  </Steps>
</div>
